linkaside.js

linkaside.js is a script you can add to a website to create a sidebar with the outgoing links on a page.

How it works

linkaside.js is a script you can add to a website.

After loading, you can use JavaScript to show or hide a sidebar with the outgoing links on a page.

You can use linkaside.js to more clearly visualize the references on a page.

Setup

First, add the linkaside.js script to your website. You can find the script in the GitHub repository.

Add the script below the closing body tag.

<script src="linkaside.js"></script>

Then, add the link-aside HTML component to your web page:

<link-aside></link-aside>
