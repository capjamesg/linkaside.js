linkaside.js

linkaside.js is a script you can add to a website to create a sidebar with the outgoing links on a page.

How it works

linkaside.js is a script you can add to a website.

After loading, you can use JavaScript to show or hide a sidebar with the outgoing links on a page.

You can use linkaside.js to more clearly visualize the references on a page.

Setup

First, add the linkaside.js script to your website. You can find the script in the GitHub repository.

Add the script below the closing body tag.

<script src="linkaside.js"></script>

Then, add the link-aside HTML component to your web page:

<link-aside></link-aside>

linkaside.js makes an API call to a compatible server that returns metadata about provided links. A hosted version at the following URL is the default:

https://jamesg.blog/lp/outgoing_links

You can also host your own API server. To do so, download the project GitHub repository and install the server:

git clone https://github.com/capjamesg/linkaside.js.git
cd linkaside.js
pip install -r requirements.txt

Then, you can run the server:

python3 app.py

The server opens one API endpoint to which you can make requests:

https://localhost:5000/outgoing_links?urls=1,2,3...

The urls parameter is a comma-separated list of URLs to which you want to get metadata.

Here is an example of a request to the API endpoint:

GET https://localhost:5000/outgoing_links?urls=https://github.com/capjamesg/linkaside.js

To configure the URL of the API endpoint, add the data-api-url attribute to the link-aside component:

<link-aside data-api-url="https://localhost:5000/outgoing_links"></link-aside>
